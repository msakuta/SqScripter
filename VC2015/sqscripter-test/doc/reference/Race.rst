Race
====

A small data container that represents statistics of a race, which is a team of Creeps.
It only has readonly properties.

.. js:attribute:: Race.id

   The race ID, which is a numeric value, starting from 0.

.. js:attribute:: Race.kills

   The total counts of Creep kills this race has done.

.. js:attribute:: Race.deaths

   The total counts of Creep losses this race has suffered.
